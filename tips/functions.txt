Useful functions for competitive programming (here i didn't put the most obvious ones, as push_back() or max())

Functions to make operations in arrays easier (can replace ispositive with a random lambda)
      all_of(first, last, ispositive()); 
      any_of(first, last, ispositive());
      none_of(first, last, ispositive()); 

Printing decimals with arbitrary precision
      cout<<fixed<<setprecision(n)<<i<<endl;

Bit manipulation
      __builtin_clz(x) (undefined behavior if x=0)
      bit_width(x) the number of bits necessary to represent x (c++ 20)

Randomize functions
      shuffle(first, last, rng) (use the rng defined in the random.hpp)
